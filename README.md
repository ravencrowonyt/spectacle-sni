# Spectacle Tray Icon

A KDE Plasma system tray icon for the Spectacle screenshot tool with quick capture modes and persistent settings.

## Features

- KDE Plasma StatusNotifierItem tray icon
- Quick screenshot capture modes
  - Rectangular region
  - Active window
  - Full screen
- Optional toggles
  - Copy to clipboard
  - Include mouse pointer
  - Include window decorations
- Remembers last used capture mode
- Persistent configuration
- Optional autostart support

Required dependencies: python python-pydbus python-gobject libdbusmenu-glib gobject-introspection spectacle
